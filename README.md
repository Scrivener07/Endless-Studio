# Endless Studio
Tooling for user created content in [Endless Space 2](http://store.steampowered.com/app/392110/Endless_Space_2/), a game by Amplitude Studios.

## Technologies
* [Microsoft .NET Framework 4.7.1](https://www.microsoft.com/net/download/windows)
* [Microsoft Entity Framework 6.2.0](https://docs.microsoft.com/en-us/ef/ef6/)
* [Microsoft Visual Studio Image Library 2017](http://www.microsoft.com/en-my/download/details.aspx?id=35825)
* [XML Schema Definition Tool](https://docs.microsoft.com/en-us/dotnet/standard/serialization/xml-schema-definition-tool-xsd-exe)
* [Entity Framework in Windows Forms, Bernardo Castilho](https://www.codeproject.com/Articles/221931/Entity-Framework-in-WinForms)
